# Primitive DB

Простая интерактивная база данных на Python с командной строкой.

## Описание

Primitive DB — это учебный проект, демонстрирующий:
- Создание Python-пакетов с Poetry
- Интерактивный CLI с библиотекой `prompt`
- Организацию кода в модули
- Публикацию пакетов на PyPI

## Установка

### Требования

- Python 3.12+
- Poetry

### Установка из исходников

```bash
# Клонировать репозиторий
git clone git@github.com:mkolesnik260592-ship-it/project-2_kolesnik_matvey_m25-555.git

cd project-2_kolesnik_matvey_m25-555

# Установить зависимости
poetry install
